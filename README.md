# 焦糖星愿泰迪 v28 FINAL · v7base

这是以用户确认的 v7 为基础继续升级的最终版，不改变 v7 的主视觉方向。

已加入 v28 AI电子宠物能力：
- 被动状态变化：饱食、快乐、清洁、亲合、困倦、孤独。
- 主动表达：饿了、想洗澡、想你、困了、想玩。
- 情绪语音：Web Speech API / 复用原语音系统。
- 记忆系统：记录喂食、洗澡、换装、玩耍、散步、故事、抚摸、小铺、背包使用偏好。
- localStorage 长期保存。
- 小铺、背包、报告、场景仍保留 v7 的视觉和结构。
- 孩子端不显示题目/知识点，学习仍隐式嵌入照顾动作。

部署：
把 index.html 放到 GitHub Pages 仓库根目录即可。
